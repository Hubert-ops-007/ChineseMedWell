# ChineseMedWell

### 中医养生大模型_internLM

书生·浦语大模型实战营大作业

**描述：**“保温杯里泡枸杞”，在繁忙的现代生活中，人们越来越重视中医养生的传统智慧，将其作为维护身心健康的重要方式。中医养生以“防病于未然，保健延年”的理念，注重调整人体的阴阳平衡、气血流畅，以达到预防疾病、延缓衰老的目的。该项目以中医养生为切入点，深挖该领域，打造中医养生帮手垂直领域大模型。
